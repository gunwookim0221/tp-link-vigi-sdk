# Code of Conduct

## Our Pledge

We aim to make participation in this project respectful, constructive, and open to people with different levels of experience.

## Expected Behavior

- Use welcoming and inclusive language.
- Respect differing viewpoints and technical tradeoffs.
- Give and receive feedback professionally.
- Focus criticism on ideas and code, not people.

## Unacceptable Behavior

- Harassment, intimidation, or discriminatory language.
- Personal attacks or sustained disruption.
- Publishing private information without permission.
- Any conduct that would make reasonable contributors feel unsafe.

## Enforcement

Project maintainers may remove comments, close issues, or restrict participation when behavior conflicts with this Code of Conduct.

## Attribution

This Code of Conduct is based on the Contributor Covenant style and adapted for this project.
